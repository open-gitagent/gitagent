Agent Engine

Agent Loop
OUTER: follow-up messages
  INNER: tool calls + steering
    1. Check steering queue
    2. transformContext() — prune/inject
    3. convertToLlm() — filter messages
    4. Stream LLM response
    5. If tool calls → execute tools
       emit: tool_start → tool_end
       check steering after each tool
    6. Create tool result messages
    7. Loop if: more calls OR steering
  8. Check follow-up queue → loop or exit

4. Hooks & Lifecycle

9. Complete Query Lifecycle

End-to-End Flow: query(options) → GCMessage stream
1. Input
prompt (string | async iterable) + options
↓
2. Load Agent
agent.yaml → resolve model → assemble system prompt
↓
3. Build Tools
builtin → declarative → injected → filtered → hook-wrapped
↓
4. Create Sandbox (if enabled)
GitMachine → E2B VM → clone repo → checkout branch
↓
5. Run Hooks
on_session_start (script + programmatic) → allow | block
↓
6. Agent Loop
new Agent(state) → subscribe(events) → prompt(text)
↓
7. LLM Streaming
model.stream() → text/thinking/toolcall deltas → AssistantMessage
↓
8. Tool Execution
pre_tool_use hook → execute → tool_result → loop back to 7
↓
9. Stream Output
AgentEvent → GCMessage → Channel → AsyncGenerator → consumer
↓
10. Cleanup
finalize session · stop sandbox · post_response hooks
